## Local Run
### Installation using miniconda
Create env.
```
conda create -n fakethermo python=3.10
conda activate fakethermo
```

Package
```
conda install flask
```

### Run
```
python thermometer.py
```

### *****
You can also set up this app using `pipenv`.
```
pipenv install
```
